# eden-app-comercios
Aplicación web que permite a los comercios crear, actualizar su perfil agregando imagenes, eventos, precios, etc.
En construcción. Hecho con Reactjs.

## Instalación

Abrir proyecto con visual studio code y ejecutar:


```react
npm i
npm run build
```

## Ejecución

```react
npm run start
```
